Sauvegarde

Les photos et avis sont enregistrés en ligne : chaque modification faite ici est immédiatement visible par tous les visiteurs, depuis n'importe quel appareil. Rien à publier.

Récupérer les photos de l'ancienne version

Si vous aviez ajouté des photos avec l'ancienne version du site, vous pouvez les envoyer en ligne : soit depuis ce navigateur, soit depuis un fichier photos.json exporté à l'époque.

Copie de secours

Sécurité

Modifiez le mot de passe de votre compte admin. Le changement s'applique partout, sur tous les appareils.

Bon à savoir : les comptes admin se gèrent dans votre tableau de bord Supabase (Authentication → Users) : vous pouvez y ajouter un second compte ou réinitialiser un mot de passe oublié.
