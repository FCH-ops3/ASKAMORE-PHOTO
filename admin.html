Publication & sauvegarde

Les photos ajoutées ici sont d'abord enregistrées dans ce navigateur. Pour qu'elles soient visibles par tous les visiteurs du site, publiez-les en 3 étapes :

1. Cliquez sur « Exporter » pour télécharger le fichier photos.json.
2. Sur GitHub, ouvrez le dossier data de votre dépôt.
3. Remplacez le fichier photos.json existant par celui téléchargé (Add file → Upload files).

Bon à savoir : l'export sert aussi de sauvegarde. Conservez une copie du fichier photos.json — vous pourrez le réimporter ici depuis n'importe quel navigateur.

Sécurité

Modifiez le mot de passe d'accès à cet espace admin.

Important : ce mot de passe est enregistré dans le navigateur utilisé. Sur un site statique (GitHub Pages), il s'agit d'une protection de confort : ne stockez jamais d'informations sensibles dans cet espace. Le mot de passe par défaut peut être changé durablement dans le fichier js/data.js (ligne DEFAULT_PASSWORD).
